# 3. What is Data Preprocessing?

Data preprocessing means:

* Cleaning data
* Transforming data
* Making data suitable for mining or machine learning

In this experiment:

* **Discretization** preprocessing is used.

---

# 4. What is Discretization?

Discretization converts:

* Numerical data
  → into categories/groups.

Example:

| Numeric Value | Converted To |
| ------------- | ------------ |
| 35°C          | hot          |
| 20°C          | cool         |

Purpose:

* Makes mining easier
* Improves understanding of patterns

---

# 5. What is Association Rule Mining?

Association Rule Mining finds:

* Hidden patterns
* Relationships between attributes

General form:

[
A \Rightarrow B
]

Meaning:

> If A happens, then B also happens.

Example:

```text id="9wzjkg"
outlook=overcast ==> play=yes
```

Meaning:

* Whenever outlook is overcast,
* People always played.

---

# 6. Algorithm Used

Algorithm: Apriori Algorithm

Apriori:

* Finds frequent item combinations
* Generates strong rules
* Uses support and confidence

---

# 9. Important Terms

---

## Support

Support tells:

> How frequently rule appears in dataset.

Formula:

Support(A \Rightarrow B)=\frac{Transactions\ containing\ A\ and\ B}{Total\ Transactions}

Example:

* Rule appears 4 times in 14 rows.

---

## Confidence

Confidence tells:

> How often rule is true.

Formula:

Confidence(A \Rightarrow B)=\frac{Support(A\cap B)}{Support(A)}

If confidence = 1:

* Rule is 100% correct.

---

## Lift

Lift tells:

> Strength of relationship.

| Lift Value | Meaning         |
| ---------- | --------------- |
| > 1        | Strong relation |
| = 1        | Normal          |
| < 1        | Weak relation   |

# Short Steps for Assignment

## Part A: Data Preprocessing

1. Open Weka
2. Click **Explorer**
3. Click **Open File**
4. Load `weather.symbolic.arff` dataset
5. Go to **Preprocess** tab
6. Click:

```text id="njm2k7"
Choose → filters → supervised → attribute → Discretize
```

7. Click **Apply**

Dataset is now preprocessed.

---

# Part B: Association Rule Mining

8. Go to **Associate** tab
9. Click:

```text id="4u0eu0"
Choose → associations → Apriori
```

10. Set parameters:

* Number of rules = 10
* Confidence = 0.9
* Minimum support = 0.1

11. Click **Start**

---

Meaning:
