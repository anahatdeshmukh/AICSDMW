# What is Clustering?

Clustering means:

# Grouping similar data together

It is:

* Unsupervised learning
* No predefined class labels required

---

# Algorithm Used

## Simple K-Means

K-Means:

* Divides data into K groups
* Groups similar instances together

Example:

```text id="c3"
K = 3
```

means:

* Dataset divided into 3 clusters.

# What Happens Internally?

WEKA:

1. Randomly selects centroids
2. Measures distances between points
3. Assigns nearest points to clusters
4. Updates centroids
5. Repeats until stable
