# Assignment No. 3

## Demonstration of Classification Process using Naive Bayes Algorithm on Iris Dataset

---

# Objective

* To perform classification using the Naive Bayes algorithm in Weka.
* To classify flower species using `iris.arff` dataset.

---

# Dataset Used

Dataset: `iris.arff`

Total:

* 150 instances
* 3 attributes after preprocessing

Attributes used:

* petallength
* petalwidth
* class

Classes:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

---

# What is Naive Bayes?

Naive Bayes is a classification algorithm based on probability.

It predicts class using Bayes theorem.

It assumes:

* All attributes are independent.

Example:

* Based on petal length and width,
* It predicts flower type.

---

# Steps Performed

1. Open Weka
2. Go to **Explorer**
3. Load `iris.arff` dataset
4. Apply preprocessing filters:

   * Remove attributes
   * Normalize
   * Replace Missing Values
   * Attribute Selection
5. Go to **Classify** tab
6. Choose:

```text id="sbhb2h"
bayes → NaiveBayes
```

7. Click **Start**
